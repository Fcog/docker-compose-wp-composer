# WordPress Docker Compose + Composer 
Set up you WordPress environment easliy with Docker Compose.

The setup includes
- mysql
- PHPMyAadmin
- composer


## Requirements
- Docker

## Installation
```bash
docker compose up
```

## Usage
Localhost: http://0.0.0.0:8000/

PHPMyAadmin: http://0.0.0.0:8080/ 

credentials: wordpress/wordpress
